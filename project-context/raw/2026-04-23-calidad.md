# Calidad

Fuente: documento compartido por el usuario el 2026-04-23.

## Objetivo general

Oracle desea mejorar la productividad y visibilidad de las actividades de cada uno de los miembros de su equipo de desarrollo en un 20%. Para ello, nuestro equipo decidió crear un portal de administración que use un Chatbot vía Telegram para automatizar y dar visibilidad a las tareas realizadas por cada uno de los miembros del equipo de desarrollo de software.

## Metas y métricas

### Disminuir el tiempo requerido para generar una tarea

- Tiempo promedio de creación de tarea (manualmente y con IA).
- Porcentaje de tareas creadas con IA.
- Cantidad de puntos de historia obtenidos durante el sprint.
- Tiempo administrativo ahorrado.

### Incrementar la cantidad de tareas realizadas

- Porcentaje de tareas con campos completos.
- Porcentaje de tareas aceptadas sin cambios.
- Porcentaje de tareas corregidas manualmente.
- Porcentaje de tareas completadas durante el Sprint.

### Mejorar la visibilidad de los proyectos de software

- Porcentaje de tareas con proyecto/prioridad definida.
- Mejor clasificación desde el momento de creación.
